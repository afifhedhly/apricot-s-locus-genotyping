# R scripts

These R scripts summarize `bedtools coverage` outputs generated by the S-locus mapping pipeline.

`01_automatic_S_genotype_calling.R`

Calls S-locus genotypes from bedtools coverage output using a minimum Breadth threshold.

`02_breadth_threshold_distribution.R`

Plots the distribution of all Breadth values and indicates the 0.50 threshold used for allele calling. The y-axis is manually split to visualize both the high number of near-zero Breadth values and the low-frequency high-Breadth calls.

`03_reference_removal_robustness_test.R`

Generates the reference-removal robustness figure comparing full-reference and modified-reference mappings.
